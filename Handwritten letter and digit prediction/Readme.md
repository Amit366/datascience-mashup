## Overview
    Hanwritten digit and letter recognition with EMNIST dataset (By-Class)
    This model predicts the handwritten letter using Convolutional Neural Network
    
    
## Dependencies
    pip install numpy
    pip install pandas
    pip install matplotlib
    pip install tensorflow
